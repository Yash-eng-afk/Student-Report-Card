# Student Report Card (Python CRUD Project)

This is a simple Python project to manage student report cards using basic file handling and JSON. It implements Create, Read, Update, and Delete (CRUD) operations in a menu-driven format.

## Features

- Add student records
- View all student records
- Search a student by roll number
- Update a student's marks
- Delete a student record
- Data is saved in a JSON file (`ReportCard.json`)

## Technologies Used

- Python 3
- JSON module
- OS module

## How to Run

1. Make sure Python is installed on your system.
2. Clone or download the repository.
3. Open the terminal or command prompt.
4. Run the program using:

```bash
python student_report_card.py
